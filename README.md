## Dependencies

- A text editor (example : [Visual Studio Code](https://code.visualstudio.com/))
- [React](https://reactjs.org/)
- [react-router-dom](https://reactrouter.com/web/guides/quick-start)
- [prop-types](https://github.com/facebook/prop-types)
